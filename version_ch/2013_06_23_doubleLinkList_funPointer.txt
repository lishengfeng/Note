4.指针函数
	函数名本身就是一个函数的首地址,函数名不可以改指向
	函数也是有类型的,函数的类型,看头(参数和返回值  ),不看函数体
	void fun1();
	void fun2();

	void (*p)();  这就能指向 void funName()类型的函数
	p1=fun1;
	p1();//调用
	p1=fun2;
	p1();//

	作用:函数指针作为参数

	int add(int a,int b);
	int sub(int a,int b);
	int Math(int a,int b,int (*p)(int,int))
	{
		return p(a,b);
