1.线性表 
	(依赖于1.内存本身就是线性的,2.逻辑上也是线性的)
	例如：数组
	优点:1.简单直观
	     2.查找速度快
	缺点:1.大小难确定
	    2.中间增加,下标改动大
            3.中间删除
2.线性表的链式存储结构(链表)
	定义结构体
	struct A{
		data;
		pointer;
	}
	例:struct studert{
		int id;
		char name[12];
		int math;
		struct student *p;
	}
	缺点:比顺序表查找速度慢
	优点:查,增,改,拓展容易

	2.1 单链表(数据域和指针域)
		指针域放后继的地址；
		最后一个指针域指针放NULL
		优缺点:找后继难,找前驱难
		有头链表和无头链表: 指针指向的第一个是有数据域就是无头的，而指向的第一个是无数据域的只有指针域的，则是无头的
